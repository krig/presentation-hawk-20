Hawk is a web interface for High Availability clusters based on
Pacemaker and corosync.

---

Installing Hawk

```bash
$ git clone git@github.com:krig/hawk-guide
$ cd hawk-guide
$ vagrant up alice
```

```bash
$ docker pull krig/hawk
```

```bash
$ zypper install ha-cluster-bootstrap
$ ha-cluster-init
```
